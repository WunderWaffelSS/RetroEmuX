Модуль Game Loader

Отвечает за загрузку, проверку и парсинг ROM-файлов.

Основные компоненты

  • rom_loader.py – Загрузка ROM-файлов.
  • bios_handler.py – Работа с BIOS-файлами.
  • format_parser.py – Парсинг форматов игр.
  • checksum_verifier.py – Проверка целостности ROM.

Функции модуля

  • Автоопределение формата ROM
  • Работа с ISO, BIN/CUE, NES, SMD
  • Загрузка и валидация BIOS

Пример кода

from game_loader.rom_loader import ROMLoader
loader = ROMLoader()
loader.load_rom("game.iso")
