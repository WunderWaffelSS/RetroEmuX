Добро пожаловать в документацию RetroEmuX

RetroEmuX – это мощный эмулятор ретро-консолей, предназначенный для запуска игр с платформ:

  • Dendy (NES, Famicom)
  • Sega Mega Drive / Genesis
  • Sony PlayStation 1/2/3
  • Sega Dreamcast

Основные модули

Проект разделен на несколько модулей:

  • Core – Логика эмуляции
  • Frontend – Графический интерфейс
  • Game Loader – Загрузка и обработка ROM-файлов

Системные требования

Для запуска RetroEmuX требуется:

  • Python 3.8 или выше
  • Модули pygame, tkinter, numpy
  • BIOS-файлы консолей (при необходимости)

Установка

git clone https://github.com/your-repo/RetroEmuX.git
cd RetroEmuX
pip install -r requirements.txt
python main.py
